# Mona Mayhem Design System

## Theme

- Retro Arcade Style
- Neon Purple & Blue Color Palette
- Dark Background
- Pixel-inspired UI
- Rounded Cards

---

## UI Rules

- Use CSS Grid where possible
- Primary Color: #7C3AED
- Secondary Color: #38BDF8
- Accent Color: #F59E0B

---

## Typography

- Bold headings
- Clean modern font
- Large CTA buttons

---

## Components

Every page should include:

- Header
- Score Panel
- Action Buttons
- Status Indicator
- Responsive Layout

---

## Animations

- Smooth transitions
- Hover effects
- Fade-in components

---

## Code Style

- Modular components
- Reusable functions
- Comments where needed
- Clean naming conventions

---

## Accessibility

- High contrast
- Keyboard friendly
- Mobile responsive
